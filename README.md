# FORUS Python Project

Учебный проект для задания по настройке рабочей среды Python-разработки.

## Описание

Проект создан для проверки настройки WSL Ubuntu, Python, Git, VS Code и GitHub.

## Структура проекта

- app/main.py — основной файл программы.
- examples/ — папка со скриншотом результата работы.
- requirements.txt — файл зависимостей.
- .gitignore — файл исключений Git.

## Запуск

python3 app/main.py
