# Работа с FastAPI на Python

Учебное задание по созданию API на FastAPI с подключением к PostgreSQL.

## Описание

Проект реализует API для работы с книгами и категориями.

## Возможности

- просмотр списка категорий;
- просмотр категории по id;
- создание категории;
- обновление категории;
- удаление категории;
- просмотр списка книг;
- фильтрация книг по категории;
- просмотр книги по id;
- создание книги;
- обновление книги;
- удаление книги;
- проверка работы сервиса через /health.

## Структура проекта

- app/main.py — точка входа FastAPI.
- app/schemas.py — Pydantic-схемы.
- app/api/categories.py — роуты для категорий.
- app/api/books.py — роуты для книг.
- app/db/db.py — подключение к базе данных.
- app/db/models.py — SQLAlchemy-модели.
- app/db/crud.py — CRUD-функции.
- app/init_db.py — создание и заполнение таблиц.
- examples/ — скриншоты результата работы.
- requirements.txt — зависимости проекта.
- .env.example — пример параметров подключения.

## Запуск

Активировать виртуальное окружение:

source venv/bin/activate

Создать и заполнить таблицы:

python3 -m app.init_db

Запустить сервер:

uvicorn app.main:app --reload

## Проверка

Swagger-документация:

http://127.0.0.1:8000/docs

Проверка состояния сервиса:

http://127.0.0.1:8000/health

Пример фильтрации книг по категории:

http://127.0.0.1:8000/books/?category_id=1
